기업 '넥스트 이노베이션'의 페이지를 클론 작업을 하면서 기존 홈페이지의 코드와 작업한 코드를 비교해 보며 다양한 방법으로 코드를 익히며 학습을 진행하였습니다.

메인페이지는 3개의 이미지가 위아래로 롤링기능을 작업하였고, 비지니스 페이지는 마우스를 내릴때마다 컨텐츠가 나오는 스크롤기능과 전광판처럼 글씨가 흘러가는 텍스트롤링 기능을 작업하였습니다.

클론사이트 바로가기

Slide Plugin

Slide Plugin을 가져와 적용하여 3장의 메인 이미지들이 마우스 휠을 내리거나 올릴 때 롤링되어 볼 수 있도록 하였습니다.

Fade In

Fade In효과를 넣어 스크롤을 내릴 때 컨텐츠들이 나타나도록 하였습니다.

Text Rolling

기존 페이지에는 Marquee을 사용하여 글자가 움직이는 효과를 주었지만, 현재는 웹 표준 Html5에서 이 기능 사용을 권장하지 않기 때문에 다른 rolling 효과를 넣어 기존 페이지와 똑같은 효과를 적용하였습니다.
