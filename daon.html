기존 '가평다온 펜션' 사이트의 다양한 문제점들을 발견하여 페이지 리뉴얼 대상으로 선정했습니다. 카테고리를 수정해 페이지를 이용하는 고객들의 동선을 최소화 할 수 있도록 하고, 이미지 컨텐츠를 최대한 활용해 다양한 시각적 정보를 제공할 수 있도록 제작했습니다.

기존 홈페이지는 대분류 6개, 소분류 18개로 이루어져 있어 펜션의 세분화된 정보를 얻는 데는 도움이 되지만 여러 번의 클릭으로 이용하는 고객들의 동선이 길어진다는 단점이 있기 때문에 이를 보완하기 위해 대분류 3개, 소분류 6개로 축소하여 동선은 짧지만 정보 전달은 정확하게 보일 수 있도록 수정하였습니다.

Image content

Slick-slider Plugin
