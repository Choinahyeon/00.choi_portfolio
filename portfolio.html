웹디자이너로서 작업했던 프로젝트들과 코딩을 배워오면서 진행했던 프로젝트들을 정리해 보는 웹페이지를 만들고자 하는 목표를 두고 제작하였습니다.

포트폴리오는 기존 작업했던 홈페이지에서는 작업해보지 않은 요소들을 넣는데 집중했습니다. Main 페이지는 스크롤만으로도 페이지의 전체적인 느낌을 볼 수 있도록 원페이지로 디자인하였고, svg를 이용한 애니메이션을 추가하였습니다. Contact페이지에는 마우스를 이용해 3D 효과를 볼 수 있는 동적인 요소들을 배치해 재미요소를 더했습니다.

Design concept

저를 간단히 소개하고 작업했던 프로젝트를 정리해놓은 페이지입니다. 좋아하는 블루를 메인 컬러로 잡고 연파랑을 보조 컬러로 잡아 디자인, 기획, 제작하였습니다. 각 페이지별로 다양한 기능과 요소를 추가해 심심하지않은 포트폴리오로 만들려했고, 누군가에게 보여지는 포트폴리오인 만큼 완성도를 높히려 노력했습니다.

One page

Main 페이지는 포트폴리오의 페이지의 전체적인 느낌을 볼 수 있도록 One page로 적용했습니다.

Count를 넣어 웹디자이너로 일했던 기간과 디자인작업했던 프로젝트를 간단히 소개했고, 웹개발과 디자인했던 프로젝트를 나열한 Portfolio, 하단엔 보신분들이 편하게 연락하고 자유롭게 Github에 접속할 수 있도록 클릭이 가능하게 작업하였습니다.

Page Navigation

Page 별 상세 설명 하단에 Navigation을 배치하여 Project 메뉴에 다시 돌아가지 않고 최소한의 동선으로도 다음 페이지로 빠르게 넘어갈 수 있도록 하였습니다.

3D Card View

전체적으로 단조로운 페이지에 명함 이미지를 만들어 마우스방향에 따라 움직이는 3D 애니메이션 효과를 넣어 재미요소를 추가하였습니다.
